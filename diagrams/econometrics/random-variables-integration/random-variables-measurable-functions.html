Random Variables as Measurable Functions

A random variable is not first a distribution. It is a measurable map. Geometry: outcomes live in Ω, values live in a target space, and observable events are pulled back by inverse images.

Map
X : (Ω, 𝓕) → (ℝ, 𝓑(ℝ))
ω ↦ X(ω)
Measurable
B ∈ 𝓑(ℝ) ⇒ X⁻¹(B) ∈ 𝓕
Information
σ(X) = { X⁻¹(B) : B ∈ 𝓑(ℝ) }
Transform
Y = g(X)
Y⁻¹(C)=X⁻¹(g⁻¹(C))

Controls

X⁻¹(B) is the region of Ω whose arrows land inside B.

Interpretation
Map: X sends each outcome ω to a numerical value. Inverse map: an observable claim about values, B, becomes an event about outcomes, X⁻¹(B).
What to notice
The inverse image is the central object: probability statements about X are always probabilities of events back in Ω.
Compact exam-style summary
Random variable.
A map X:Ω→ℝ such that Borel sets pull back to measurable events.
Probability law.
P(X∈B)=P(X⁻¹(B)). The distribution is the pushforward of P by X.
Random vector.
X:Ω→ℝᵈ is measurable when inverse images of Borel sets in ℝᵈ are in 𝓕.
Information.
σ(X) is the smallest σ-algebra making X observable.
