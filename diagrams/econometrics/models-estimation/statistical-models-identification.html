Statistical models and identification

A parametric model is a map from parameters into the space of probability laws. Identification means this map does not collapse distinct parameters into the same distribution.
Model
𝓟 = {Pθ : θ ∈ Θ} a curve inside the probability simplex
Identification
Pθ = Pθ′ ⇒ θ = θ′ the parameter-to-law map is injective
True parameter
P0 = Pθ₀ only under correct specification
Pseudo-true
θ★ ∈ argmin KL(P0||Pθ) closest model law when \(P_0\notin\mathcal P\)

Interpretation
Model.
\(\Theta\) is not the space of distributions. It is mapped into the space of distributions.
Identification.
Failure occurs when distinct parameters land at the same point in the simplex.
True parameter.
If \(P_0\) lies on the model curve and the preimage is unique, \(\theta_0\) is identified.
Pseudo-true.
If \(P_0\) lies outside the curve, estimation targets the KL projection \(P_{\theta★}\).
