Distribution families

Anatomy of a distribution family

A parametric family is a geometric surface of probability laws. A single parameter value selects one law; changing location, scale, shape, or support moves through the family.

    Compact exam summary

    Minimal facts to identify a distribution family.

    1

    State Θ. A parameter value is meaningful only after its admissible set is specified.

    2

    State support. Check whether the support depends on θ; this changes likelihood geometry.

    3

    Check identifiability. Different coordinates must not represent the same law.

    4

    Separate roles. Location shifts, scale stretches, shape bends, support restricts.
