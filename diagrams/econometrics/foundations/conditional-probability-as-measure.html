07 · Conditional Probability as a Measure

Conditioning does not merely change a number. It changes the sample space: once information B is accepted, probability is re-measured inside B.

Definition
PB(A) = P(A | B) = P(A ∩ B) / P(B),   P(B) > 0
New space
ΩB = B
FB = { A ∩ B : A ∈ F }
Measure
PB(B) = 1
PB(⋃ Ci) = Σ PB(Ci)
Null condition
P(B)=0 ⇒ P(A | B) is not defined by the ratio.

Interpretation
What to notice
Compact exam-style summary
1. If P(B) > 0, then A ↦ P(A | B) is a probability measure on F.
2. Conditioning replaces Ω by B. The relevant event is A ∩ B, not A alone.
3. P(A | B)=1 means B \ A is null. P(A | B)=0 means A ∩ B is null.
4. If P(B)=0, the elementary ratio has denominator zero. One needs a limiting rule or a regular conditional probability.
