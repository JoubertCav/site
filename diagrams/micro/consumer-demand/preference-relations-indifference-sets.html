Preference relations and indifference sets

A consumption set is the feasible universe. A preference relation orders bundles inside it. Upper and lower contour sets are the geometric shadows of the relation around a reference bundle.

weak preference: x ≽ y strict preference: x ≻ y indifference: x ∼ y upper contour: U(x̄) lower contour: L(x̄)

Main diagram

The colored regions are sets in the consumption space, not areas of utility.
U(x̄) = {x ∈ X : x ≽ x̄} L(x̄) = {x ∈ X : x̄ ≽ x} I(x̄) = {x ∈ X : x ∼ x̄}

Symbolic definition

A binary relation on the consumption set.
Consumption set
X = ℝ²₊
Preference relation
x ≽ y ⇔ u(x) ≥ u(y)
Derived relations
x ≻ y ⇔ x ≽ y and not(y ≽ x)
x ∼ y ⇔ x ≽ y and y ≽ x
Contour sets at x̄
U(x̄) = {x ∈ X : x ≽ x̄}
L(x̄) = {x ∈ X : x̄ ≽ x}
I(x̄) = U(x̄) ∩ L(x̄)
For monotone preferences, the upper contour set lies northeast of the indifference curve.

ObjectGeometric meaning
U(x̄)All bundles weakly preferred to the reference bundle. For utility-represented preferences: u(x) ≥ u(x̄).
L(x̄)All bundles that x̄ weakly dominates. For utility-represented preferences: u(x) ≤ u(x̄).
I(x̄)Bundles mutually weakly preferred to x̄. It is the boundary between better and worse only in well-behaved complete preferences.
Incomplete caseSome bundles may be incomparable: neither z ≽ x̄ nor x̄ ≽ z. Then z belongs to neither contour set.

What to notice

The same definitions generate different geometries.
Axioms currently illustrated
A preference relation is primitive. A utility function is only a representation: it assigns numbers preserving the order. The geometry of contour sets is often more fundamental than the utility numbers themselves.

Compact exam-style summary

Minimal checklist for preference relations and indifference sets.

1. Start from X

The relation is defined only on the consumption set X. If z ∉ X, the preference comparison is not an admissible comparison inside the model.

2. Define the relation

Weak preference is primitive. Strict preference and indifference are derived from weak preference, not separate assumptions.

3. Read contours

Upper contour means “at least as good as x̄.” Lower contour means “no better than x̄.” Indifference is their intersection.
