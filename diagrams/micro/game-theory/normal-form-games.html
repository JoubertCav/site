\(s_i^*\in BR_i(s_{-i}^*)\)

\(s_i\succ s_i'\)

Interpretation

Normal form. Timing is suppressed. What matters is the set of strategies and the payoff assigned to every strategy profile.
Best responses first. Nash equilibrium is not “highest joint payoff.” It is mutual best response.
Dominance is prior. A strictly dominated strategy is never optimal against any opponent strategy, so it can be eliminated before searching for equilibrium.

Exam-style summary

1
Write the payoff matrix carefully: row payoff first, column payoff second.
2
For each column, mark the row player's best response.
3
For each row, mark the column player's best response.
4
A cell with both marks is a pure Nash equilibrium.
5
Check dominated strategies: if one strategy is always worse, eliminate it.
