Intermediate Value Theorem

A continuous function on one connected interval cannot skip an intermediate height. Visually: a curve joining two endpoint heights must cross every horizontal level between them.

Controls theorem case

Visual reading Move the horizontal level. In the theorem case, every intermediate level is forced to meet the curve.

Symbolic core

\[ f:[a,b]\to\mathbb R \text{ continuous},\qquad y_0 \in [f(a),f(b)] \Longrightarrow \exists c\in[a,b]: f(c)=y_0 . \]

Sign-change form

\[ f(a)f(b)<0 \Longrightarrow 0\in(f(a),f(b)) \Longrightarrow \exists c: f(c)=0 . \]

What the theorem is saying

The domain is one unbroken interval; the graph is unbroken; therefore the image has no missing height between the endpoint heights.

What to notice

No gap in the domain + no jump in the function The image of the interval is again an interval, so every height between endpoint values is included.

Exam summary

  1. Check the hypotheses: compact interval is not essential for IVT; connected interval and continuity are.
  2. To prove a root, show \(f(a)\) and \(f(b)\) have opposite signs.
  3. The topological statement is: continuous images of connected sets are connected.
  4. In \(\mathbb R\), connected sets are intervals. Therefore the image contains all intermediate values.
