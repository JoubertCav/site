    Metric proof

    Verify the four axioms. Triangle inequality is usually the only nontrivial step.

    Same topology?

    Different formulas can induce the same open sets, as Euclidean, taxicab, and sup metrics do on Rⁿ.

    Non-example test

    A distance-looking formula fails if one axiom fails, even if it behaves well in many cases.

    Metric geometry is the visible part of the axioms: balls → neighborhoods → convergence → topology.
