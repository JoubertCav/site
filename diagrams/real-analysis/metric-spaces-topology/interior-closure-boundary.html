Metric-topology definitions

Br(x) = { y ∈ ℝn : d(x,y) < r }
x ∈ int(A) ⇔ ∃r>0 : Br(x) ⊂ A
x ∈ cl(A) ⇔ ∀r>0 : Br(x) ∩ A ≠ ∅
x ∈ bd(A) ⇔ ∀r>0 : Br(x) sees A and ℝn \ A

What the topology is doing

1
Balls are the primitive test objects.Changing the norm changes the shape of balls, but on finite-dimensional ℝⁿ the usual norms generate the same open sets.
2
Interior means stable membership.If x is interior, small perturbations of x remain in A.
3
Closure means arbitrarily good approximation.x belongs to cl(A) when points of A can be found inside every neighborhood of x.
4
Boundary means unavoidable ambiguity.Every neighborhood intersects both A and its complement.

Compact exam-style summary

int(A) = {x : ∃r>0, Br(x) ⊂ A}
cl(A) = {x : ∀r>0, Br(x) ∩ A ≠ ∅}
bd(A) = cl(A) \ int(A)
x ∈ A′ ⇔ ∀r>0, (Br(x) \ {x}) ∩ A ≠ ∅
A dense in ℝn ⇔ cl(A)=ℝn
x isolated in A ⇔ ∃r>0, Br(x) ∩ A = {x}
