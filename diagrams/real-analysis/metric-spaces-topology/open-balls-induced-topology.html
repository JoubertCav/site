Open balls and induced topology

A distance function draws balls. Balls define neighborhoods. Neighborhoods define open sets. The collection of all open sets is the topology generated by the metric.

Discrete metric

topology changes
d(x,y)=0 if x=y, and d(x,y)=1 if x≠y
For 0<r≤1, Br(x)={x}. Hence every subset is a union of singleton balls, so every subset is open.

Definitions

Br(x) = { y : d(x,y) < r }
G open ⇔ ∀x∈G, ∃r>0 : Br(x) ⊂ G
N neighborhood of x ⇔ ∃r>0 : Br(x) ⊂ N

What to notice

Open is a local property: check each point using a sufficiently small ball.
Boundary points kill openness when every ball leaks outside.
Equivalent metrics may draw different balls but generate the same family τd.
The discrete metric makes every subset open; topology can be much finer than Euclidean topology.
