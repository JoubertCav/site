Real analysis · Topology of Rn

Open versus closed sets

A set in Rn is open when every point carries a small ball staying inside. It is closed when it contains all its sequential limit points — equivalently, when its complement is open.

1. The local test in Rn

A open ⇔ for every x ∈ A, there is r > 0 such that Br(x) ⊂ A. A closed ⇔ whenever xk ∈ A and xk → x in Rn, then x ∈ A. Br(x) = { y ∈ Rn : ‖y − x‖ < r }.

2. Complement duality

A is open ⇔ Rn \ A is closed. A is closed ⇔ Rn \ A is open.

3. Clopen sets in Rn

For n ≥ 1, Rn is connected. Therefore the only clopen subsets of Rn are ∅ and Rn.

4. Finite versus arbitrary operations

The topology of Rn is built to be stable under arbitrary unions and finite intersections of open sets. By complement duality, closed sets are stable under arbitrary intersections and finite unions.

What to notice

Open is a local property: every point of A has room to move inside A.

Closed is a limit property: A contains the limits forced by sequences in A.

A set can be open, closed, both, or neither. In Rn, only ∅ and Rn are both open and closed.
