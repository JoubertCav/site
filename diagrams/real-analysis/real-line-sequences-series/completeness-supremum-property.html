The real line is not merely ordered. It has no missing boundary points: every nonempty bounded-above set has a least upper bound.

What to notice

Exam summary

To prove s = sup A, prove:

1. ∀x∈A, x≤s.
2. ∀ε>0, ∃x∈A : s−ε < x ≤ s.

The second line says: no smaller number can still bound the set.
