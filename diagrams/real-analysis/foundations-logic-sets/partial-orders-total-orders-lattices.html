Partial Orders, Total Orders, and Lattices

A partial order is a geometry of comparison. A lattice is the special case where every two points have a best common upper bound and a best common lower bound.

comparability:   x ⪯ y or y ⪯ x upper bound:   x ⪯ u and y ⪯ u x ∨ y:   least element of all upper bounds x ∧ y:   greatest element of all lower bounds

What to notice

    Compact exam-style summary

    • Partial order: reflexive, antisymmetric, transitive.
    • Total order: partial order + every pair is comparable.
    • Maximum: one element above all others. Maximal: no strictly larger element.
    • Supremum: least upper bound, not merely a minimal upper bound.
    • Lattice: every pair has both a supremum x ∨ y and an infimum x ∧ y.

    Visual dictionary

    • Edges upward: immediate order steps, called covers.
    • Chain: a path where all chosen elements are mutually comparable.
    • Antichain: a set of mutually incomparable elements.
    • Join: the first common point above two elements.
    • Meet: the first common point below two elements.
