What to notice

The universe is fixed. Complements are always relative to U, not to the whole plane.
Membership is logical. x ∈ A ∪ B means A(x) ∨ B(x).
Laws are pointwise. Two sets are equal iff every point has the same truth value on both sides.
Atoms explain everything. If each atom is classified identically, the identities hold globally.

Compact exam summary

A ∪ B = {x ∈ U : A(x) ∨ B(x)},   A ∩ B = {x ∈ U : A(x) ∧ B(x)}.
Aᶜ = U ∖ A = {x ∈ U : ¬A(x)},   A ∖ B = A ∩ Bᶜ.
A △ B = (A ∖ B) ∪ (B ∖ A) = {x ∈ U : A(x) ⊕ B(x)}.
De Morgan: (A ∪ B)ᶜ = Aᶜ ∩ Bᶜ,   (A ∩ B)ᶜ = Aᶜ ∪ Bᶜ.
Distributivity: A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C),   A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C).
Method of proof: take arbitrary x ∈ U, translate membership into logic, simplify the proposition, translate back into sets.
