Real Analysis · Set size

Countability and enumeration

A set is countable when its elements can be listed without missing any. The geometry is not size in space, but the existence of an ordering path.

Symbolic core

Enumeration means replacing a set by an ordered list indexed by natural numbers.

finite:   A ≅ {1,…,m} countably infinite:   A ≅ ℕ at most countable:   ∃ surjection e: ℕ → A uncountable:   no enumeration e: ℕ → A exists
list = surjection ℕ → A perfect list = bijection ℕ ↔ A diagonal = path through two indices

Cantor diagonal

A proposed list of binary sequences creates a new sequence not on the list.

The displayed table is finite, but the proof is not. For any infinite list x¹,x²,… define yₙ = 1 − xⁿₙ. Then y differs from row n at coordinate n.

What to notice

1A countable set is not “small” geometrically; it is listable.
2ℕ×ℕ is countable because every diagonal i+j=s is finite.
3Countable unions use the same grid idea: row index + position in row.
4Cantor diagonal proves failure of every possible list, not just the visible one.

Exam-style summary

FFinite: A ≅ {1,…,m}.
CCountable: A finite or A ≅ ℕ; equivalently A is the range of a sequence.
×ℕ×ℕ is countable by diagonal enumeration.
∪A countable union of countable sets is countable.
D{0,1}ℕ is uncountable by Cantor diagonalization.
R[0,1] and ℝ are uncountable because binary/decimal sequences cannot be fully enumerated.
The object is not cardinality as magnitude. The object is the existence or impossibility of a listing rule.
