Diagrams / Real Analysis

Real Analysis for Economists.

Filtered diagram topics from the book's table of contents: key theorems, structural maps, and economic applications.

Parts

Diagram folders by part

Each folder keeps the part-chapter-section structure and lists only topics that are useful as visual diagrams.

Part I

Logic and Proof Theory

Logical equivalences, induction, well-ordering, and proof forms used throughout analysis.

 Part II

Set Theory and Structures

Axioms, relations, orders, functions, cardinality, countability, and preference structures.

 Part III

The Real Number System

Ordered fields, completeness, density, sequences, compactness of subsequences, and series.

 Part IV

Classical Analysis in R^n

Topology, continuity, differentiation, integration, optimization, and economic comparative statics.

 Part V

Metric and Topological Analysis

Metric spaces, compactness, completeness, uniform convergence, correspondences, and maximum theorems.

 Part VI

Linear and Functional Spaces

Vector spaces, normed spaces, Banach spaces, Hilbert spaces, projections, and fixed point theory.

 Part VII

Convex Analysis

Convex sets, convex functions, separation, support, duality, minimax, and shadow prices.

 Part VIII

Measure and Integration

Measure spaces, Lebesgue integration, convergence theorems, Lp spaces, and Radon-Nikodym.

 Part IX

Economic Frontiers

Dynamic programming, optimal control, Arrow-Debreu equilibrium, welfare theorems, and games.
